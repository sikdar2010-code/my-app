# My App 🚀

A full-stack web application.

## 📁 Project Structure

- frontend → React / Next.js
- backend → Node.js / Express

## ⚙️ Setup Instructions

### 🔹 Backend

cd backend
npm install
npm run dev

### 🔹 Frontend

cd frontend
npm install
npm run dev

## 🌐 Features

- User authentication
- API integration
- Full-stack architecture
